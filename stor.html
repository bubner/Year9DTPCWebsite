INFORMATION 
                A solid-state drive (SSD) is a type of solid-state storage that, in the hierarchy of computer storage,
                serves as long-term permanent storage and uses integrated circuit assemblies to store data persistently,
                generally utilising flash memory. There are two common types of SSD, which run on two protocols, SATA and M.2.
                M.2 is the fastest and most reliable among the two, whereas SATA is still reliable, but compared to M.2 it falls behind
                in speed ever so slightly. Compared to the other form of storage, HDDs, which use magnetic disk storage, both of these
                SSDs are superior in performance compared to them has they do not use any moving parts.
DESCRIPTION 
            Each of these components share similar characteristics with eachother. Both components host high speed data read and write speeds,
            which are essential for game files to run from, while the SSDs are more designed for files such as game clips and other large files/game archives.
            Faster read and write performance on drive will improve game load speeds, operating system file load speeds, and
            general performance on the computer. In contrast, the SSDs are rated for high reliability, for the storage of large files and for personal files that
            should be kept in a safe and reliable manner.
